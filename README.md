# 🚀 My Little Cookbook
> A web application to track recipes


## What Is This?
With this app, you can keep track of recipes found from different sources. It works especially well with the ones found online.


## Why I Built It
When I moved from Paris to Munich, I lost a majority part of my recipe inventory due to the logistics. 
The reicpes were mostly found online from different sources, which made it difficult to trace back.
Then I had an idea about an online recipe book where I could simply copy and paste the ones I liked and 
nver worry about the rest.


## Key Features
- ✅ Feature 1 — create, read, update and delete recipes, including photos
- ✅ Feature 2 — mark recipes as likes
- ✅ Feature 3 - share recipes with other on social media
